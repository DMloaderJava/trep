# Trep

Социальная сеть на TanStack Start + Supabase

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

## 🛠 Технологии

- **Фреймворк:** TanStack Start
- **База данных:** Supabase
- **Авторизация:** Supabase Auth  
- **UI:** shadcn/ui
- **Сборка:** Vite

## 📁 Структура проекта

```
src/
├── components/    # UI компоненты
├── hooks/         # React хуки
├── integrations/  # Интеграции (Supabase)
├── lib/           # Утилиты и API функции
└── routes/        # Страницы (TanStack Router)
```
